项目配置

@编译器：MyEclipse 版本号不清除哪里看，2018年之后的估计都可以
@编程语言：Java 、 Html 、 JavaScript(少量)
@JDK版本：1.8之后
@服务器：MyEclipse自带服务器 Tomcat7.0 8.0 也可以
@Jar包：commons-fileupload-1.3.2.jar 文件上传下载 包内带有
     commons-io-2.5.jar 文件处理 包内带有
     mysql-connector-java-5.1.13-bin.jar 数据库连接 包内带有
@数据库：Wampserver自带的phpmyadmin 我已经导出为 .sql 结尾的文件了， 自己百度导入或者重建


注意事项

  1、我并未将图片放入数据库当中，图片的路径也是固定的，数据库未存储图片相关数据
  2、图片存储路径为服务器的虚拟路径，每次服务器重启之后原先虚拟路径内的图片会消失，会导致客户端图片为损坏状态，可以将路径改为服务器主机的真实路径
  3、项目所要构建的运行环境比较复杂，新手建议先学习一下 wampserver的用法
  4、我还没有想好。。。
